# 📈 TradeSphere – Intelligent Stock Portfolio Management Platform

TradeSphere is a full-stack stock portfolio management platform built using React.js, Node.js, Express.js, and MongoDB. The application enables users to securely manage stock portfolios, place buy/sell orders, monitor holdings, and analyze portfolio performance through an interactive dashboard.

---

## 🚀 Features

- Secure User Registration & Login (JWT Authentication)
- Password Encryption using bcrypt
- Buy & Sell Stock Orders
- Holdings Management
- Order History
- Positions Tracking
- Interactive Portfolio Analytics
- Stock Search Watchlist
- Responsive Dashboard
- MongoDB Atlas Integration
- RESTful API Architecture

# 🔗 REST API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| POST | `/signup` | Register User |
| POST | `/login` | Authenticate User |
| GET | `/allHoldings` | Fetch Holdings |
| POST | `/newOrder` | Place Buy/Sell Order |
| GET | `/allOrders` | Fetch Order History |
| GET | `/allPositions` | Fetch Positions |

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Nitish-04P/TradeSphere.git
```

Install Backend

```bash
cd backend
npm install
npm start
```

Install Dashboard

```bash
cd dashboard
npm install
npm start
```

Install Frontend

```bash
cd frontend
npm install
npm start
```
